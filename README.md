# Excel Loan Analysis Dashboard

##  Project Overview
This project is an interactive Excel Dashboard built to analyze loan data using Pivot Tables, Pivot Charts, Slicers, and Excel formulas.

##  Dashboard Features

- Total Sales
- Total Requested Amount
- Count of Cases
- Total Amount Not Approved
- Total Sales per Branch
- Total Sales by Product
- Total Cases per Month
- Product & Year Slicers

##  Tools Used

- Microsoft Excel
- Pivot Tables
- Pivot Charts
- Slicers
- Dashboard Design
- Number Formatting

##  Key Insights

- Auto Loan generated the highest sales among all products.
- March recorded the highest number of loan cases.
- Branch performance varied significantly across years.
- A large portion of the requested amount was not approved, indicating strict approval criteria.
